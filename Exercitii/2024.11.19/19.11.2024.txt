1. Write a Java program to create an array list, add some colors (strings) and print out the collection.
1.2 Write a Java program to iterate through all elements in an array list.
1.3 Write a Java program to insert an element into the array list at the first position.
1.4 Write a Java program to retrieve an element (at a specified index) from a given array list.
1.5 Write a Java program to update an array element by the given element.
1.6 Write a Java program to sort a given array list.
1.7 Write a Java program to reverse elements in an array list.
2. Creează un HashSet de tip Integer care să conțină valorile 10, 20 și 30. Afișează toate elementele utilizând un buclă FOR-EACH.
3. Creează un HashMap care să stocheze perechi de tipul <Nume, Vârstă>. Adaugă trei perechi: ("Ana", 25), ("Ion", 30), ("Maria", 28). Iterează prin HashMap și afișează fiecare pereche sub formă de text.
4. Creează o aplicație care gestionează un catalog al studenților. Fiecare student are un nume și o listă de note. Utilizează o structură de tip HashMap pentru a stoca perechi de tipul <String, ArrayList<Integer>>. Adaugă câțiva studenți cu notele lor, calculează media fiecărui student și afișează-o.
5. Creează un HashSet care stochează numere întregi. Scrie o metodă care primește setul ca parametru și returnează un nou HashSet care conține doar numerele pare.
6. Creează un ArrayList de tip String care conține mai multe cuvinte. Folosește un Iterator pentru a elimina toate cuvintele care conțin mai puțin de 4 caractere.
7. Creează o aplicație care utilizează un HashMap pentru a reprezenta un inventar de produse. Cheia va fi numele produsului, iar valoarea va fi cantitatea stocată. Adaugă o metodă pentru a:
- Incrementa cantitatea unui produs.
- Decrementa cantitatea unui produs, dar doar dacă rezultatul nu devine negativ.
- Afișa produsele și cantitățile lor.

Tema:
1. Creează un TreeMap care stochează notele obținute de studenți la un examen. Fiecare intrare din TreeMap conține perechea <nume_student, nota>. Afișează în ordine alfabetică lista cu studenți și notele lor. Adaugă și o metodă pentru a găsi nota maximă și numele studentului care a obținut-o.
2. Creează o aplicație care gestionează un magazin online folosind un HashMap pentru stocarea produselor și prețurilor lor. Adaugă funcționalități pentru:
- Adăugarea unui produs nou.
- Ștergerea unui produs.
- Găsirea produsului cu cel mai mare preț.
- Afișarea tuturor produselor și a prețurilor lor.
3. Scrie un program care primește o propoziție de la utilizator și afișează toate cuvintele unice din propoziție folosind un HashSet. Ignoră diferențele de majuscule/minuscule.
4. Creează o aplicație care simulează gestionarea unei biblioteci. Folosește un HashMap pentru a păstra cărțile (titlu carte -> nume autor). Adaugă funcționalități pentru:
- Adăugarea unei cărți.
- Găsirea autorului unei cărți.
- Ștergerea unei cărți.
- Afișarea tuturor cărților din bibliotecă.
